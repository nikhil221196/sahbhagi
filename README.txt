# Institute Chat App (Flutter + Firebase)

## How to Setup Before Build

1. Go to Firebase Console: https://console.firebase.google.com/project/sahbhagi-663d0/settings/general/
2. Scroll to **Your Apps** → Click Android icon.
3. Package name: com.institute.chat
4. Download `google-services.json` and place it here:
   android/app/google-services.json
5. Upload this ZIP to Codemagic → Select Flutter Android build → Start build.
6. After 5-10 minutes, Codemagic will give you the APK file.
